# ניתוח סיכוני אשראי

פרויקט זה מבוסס על מערך נתוני סיכוני אשראי שמציג היסטוריית הלוואות של לקוחות. מטרת הפרויקט היא לזהות לווים בעלי סיכון גבוה לחדלות פירעון ולחזות את יכולת ההחזר של הלוואות בעתיד.

## מטרות הפרויקט
- זיהוי הפרופיל של לווה מסוכן.  
- ניתוח השפעת מאפיינים שונים על סיכון ההלוואה:
  - היסטוריית חדלות פירעון
  - גיל הלווה
  - יחס החזר מההכנסה מול ותק תעסוקתי
  - סכום ההלוואה
  - דירוג ההלוואה

## כלי עבודה
- **Python (Pandas):** ניקוי נתונים, מילוי ערכים חסרים, יצירת עמודות חדשות ופיצול הנתונים לגיליונות.  
- **SQL:** שאילתות מורכבות לניתוח דפוסים בקבוצות גיל, מטרות הלוואה ודירוגים.  
- **Tableau:** יצירת גרפים ודשבורדים להצגת התובנות בצורה ויזואלית.

## מסקנות עיקריות
לווים בסיכון גבוה הם אלו שהגיעו בעבר לחדלות פירעון, בגילאי 20–29 או מעל גיל 60, שלוקחים הלוואה לאיחוד חובות, בעלי ותק עבודה נמוך מ‑4 שנים ודירוג ההלוואה שלהם D ומטה.
